项目说明：
（a）项目信息：
        背景：我们发现美团网美食类店铺评论区原有的关键词都是一样的（比如味道赞，服务热情等），不够个性化，针对性不强。绝大多数商家的5分好评或者1分差评中有很多是不具有参考价值的，并且美团的评分机制有缺陷，计算的并不是总的平均分。
        项目详情：该项目选取美团网上武汉日韩料理排行榜中的店铺为例，对其详情及评论进行爬取并对有效评论进行情感极性分析，获得实际情感倾向较高和较低的店铺信息，生成个性化的店铺标签词云图。利于了解优秀商家的优点，并且在用户选定需求后对其进行简单的推荐。
 

（b）完成情况：爬取商家详情和评论基本完成；
                词云图的功能基本实现，但因为分词及词云图的显示特点（词云图里不显示单个 字符）“不好吃”可能会显示为“好吃”，但是利用停用词和自加词典后还是可以体现出一定的商家特色及主要印象； 
                推荐程序用已爬取得到的店铺分类，用户选择“韩式烤肉""日式寿司”等类别时，推荐相应店铺。
                 情感分析可以实现积极和消极两类的极性分析，准确率在86%左右，（5分好评有一些浑水摸鱼的情况，不体现情感或情感负向但给分为5，影响准确率）有效帮助用户了解商家实际高分比例和低分比例。但是训练数据集不够丰富，准确率不够高，并且目前无法对20~40分的评论做出多分类的情感分析。
              总体衔接的不够好，还应该把各个模块整合得更完善。
